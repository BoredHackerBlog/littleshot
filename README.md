# Running the project
## requirements
- Docker - easy way to install: https://get.docker.com/
- Docker-compose - https://docs.docker.com/compose/install/

```
git clone https://github.com/BoredHackerBlog/littleshot
cd littleshot
docker-compose up --build -d caddy
```
